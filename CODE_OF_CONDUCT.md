# コントリビューター行動規範（Code of Conduct）

本プロジェクトでは、参加者全員にとって安全で歓迎される環境を維持するため、
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) に準拠した行動規範を採用しています。

## 要約（日本語）

- **歓迎されるふるまい**
  - 他者への共感と思いやりを示す
  - 異なる意見・経験・視点を尊重する
  - 建設的なフィードバックを行い、また受け入れる
  - 自分の誤りに対して責任を持ち、影響を受けた人に謝罪し学ぶ
  - コミュニティ全体にとって最良となる判断を優先する

- **容認されないふるまい**
  - 性的な表現・誘いかけ、性差別的な発言
  - トローリング、侮辱・中傷、人格攻撃、政治的攻撃
  - 公的・私的なハラスメント
  - 他者の住所、メール、その他の個人情報を許可なく公開する
  - その他、プロフェッショナルな場で不適切と見なされる行為

## 適用範囲

本行動規範は、プロジェクトに関わるすべての公的空間（GitHub Issue / Pull Request / Discussion / コミットメッセージ / コードコメント / 関連するチャットやメール）に適用されます。
個人が本プロジェクトを公式に代表している場面（公式アカウントでの発信、登壇など）も含みます。

## 違反の報告

行動規範違反を目撃した、または受けた場合は、以下の窓口に報告してください。
報告内容は秘密に取り扱い、報告者の意向を尊重します。

- 連絡先: `conduct@syncpulse.example.com`

セキュリティ脆弱性は別フローのため、[`SECURITY.md`](./SECURITY.md) を参照してください。

報告には以下を含めると対応が迅速になります:

- いつ・どこで・誰が関わった出来事か
- 関連するスクリーンショット・リンク・ログ
- 報告者として希望する対応（謝罪・モデレーション・公表の有無 等）

## 対応方針

メンテナは報告された違反を確認し、状況に応じて以下のいずれかを段階的に適用します。
詳細は Contributor Covenant v2.1 の Enforcement Guidelines に準拠します。

1. **訂正**: 非公開での注意。望ましい行動の説明。
2. **警告**: 一定期間、対象者との関わりを制限するよう要請。
3. **一時的な追放**: コミュニティ空間からの一時的なアクセス禁止。
4. **恒久的な追放**: 重大または繰り返される違反に対する恒久的なアクセス禁止。
